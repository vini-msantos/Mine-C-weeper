## Dependencies:
To run the project you must have both GCC and Makefile installed.

## Installation guide:
1. Clone the git project:
```bash
git clone https://github.com/vini-msantos/Mine-C-weeper
```
2. Change the working directory to the downloaded folder:
```bash
cd Mine-C-weeper
```
3. Run the following command to execute the makefile:
```bash
make
```
4. Finally, run the generated executable with:
```
./minecweeper
```
